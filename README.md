# Customer Sales Dashboard

## Project Overview
This project analyses retail sales data using Python, Pandas and Matplotlib.

The objective is to identify sales trends, customer purchasing behaviour and product performance to support business decision-making.

## Dataset
The dataset contains:
- Customer information
- Product categories
- Quantity sold
- Unit prices
- Total sales
- Transaction dates

## Technologies Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Project Tasks
- Load the dataset
- Inspect the data
- Generate summary statistics
- Check for missing values
- Analyse sales by product category
- Visualise business insights
- Create an executive dashboard

## Key Business Insights
- Clothing generated the highest revenue.
- Electronics followed closely behind.
- Beauty products generated the lowest revenue.
- The executive dashboard summarises key business performance indicators.

## Executive Dashboard
The dashboard displays:
- Total Revenue
- Total Transactions
- Total Customers
- Average Sale Value
